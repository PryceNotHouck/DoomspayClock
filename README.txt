For the project grader to run the program:
1. Download the 'For Grader.zip' file from the landing page on the repository.
2. Extract, and open the 'Ship' folder into CLion or any other preferred C++ environment.
3. Remove 'Clock UI' from the first line of the executable in CMakeLists, (CLion) right click the CMakeLists file in the project explorer and select the option to load that configuration.
4. Run 'Clock UI Executable.cpp' by running the loaded configuration.
